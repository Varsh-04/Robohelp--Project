HubSpot

 

Segments

You can create and maintain all segments, calls, and meeting links in HubSpot.

Segments

You can create segments to save filters that are used repeatedly to companies, tickets, issues, tasks etc...

 

 figure 1

 

  • Press the Menu button

 

figure 2

 

  • Press the Segments button

List of all the segments will display

 

Type

 

You can view segments that are applied to each sections individually.

 

figure 3

 

  • Press the Menu button

  • Press the Segments button

  • Press Object Type drop-down, and then select type

A list of segments related to the type will display

 

Calls

 

You can view the list of all the recorded, assigned calls, and transcript the calls to text.

Note: The recorded calls will display only if the calls are made using HubSpot and recorded

 

  • Press the Menu button

 

figure 4

 

  • Press the Calls button

List of all the recorded calls will display

 

Assigned calls

 

You can filter and view the calls assigned to yourself or any other person.

 

  • Press the Menu button

  • Press the Calls button

 

       figure 5

 

  • Press the Assigned to button

  • Select people, and then press the Apply button

A list of all assigned calls for the selected person will display

 

Transcript calls

 

Transcript calls refer to calls that have been converted to text. You can view list of all transcripted calls.

 

  • Press the Menu button

  • Press the Calls button

 

   figure 6

 

  • Press the Transcript All button

List of all the transcripted calls will display

 

Enable calling

 

To call, you must enable calling option in HubSpot.

 

  • Press the Menu button

 

figure 7

 

  • Press the Settings button

 

figure 8

 

  • Press the Calling button

 

figure 9

 

  • Press the Call method button

 

figure 10

 

  • Press the HubSpot Calling button

After enabling, you can make calls through HubSpot

 

Make and record calls in HubSpot

 

You can call a person through HubSpot.

 

  • Press the Menu button

  • Press the Contacts button

List of all the contacts will display

 

figure 11

 

  • Select a contact to make a call

 

figure 12

 

  • Press the Call button

  • Select call method as HubSpot

  • Press the Record button

Recording will start and will be saved in calls section in HubSpot

 

Meeting Links

 

In meeting links, you can view the list of meetings links that you created.

Note: You can create meeting links only in desktop

 

  • Press the Menu button

  • Press the Meeting links button

List of all the meeting links will display
